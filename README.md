# AI Sector Intelligence

基于 Next.js、Supabase 与 DeepSeek 构建的 AI 板块情报平台。

项目通过自动采集产业资讯，利用大模型完成板块分类、情绪分析与摘要生成，并生成结构化行业情报，帮助投资者快速了解 AI、机器人、算力、半导体、新能源等热门赛道的市场动态。

## 在线演示

https://ai-sector-intel-v2-ioqk.vercel.app

---

## 项目背景

行业资讯广泛分布于新闻网站、企业公告、行业媒体及政策文件中，信息来源分散且更新频繁。

对于普通投资者而言，获取有效信息往往需要投入大量时间，同时还需要自行判断新闻所属板块及其潜在影响。

本项目尝试利用大模型自动完成新闻理解与结构化处理，构建轻量级行业情报系统，实现从资讯获取到板块分析的自动化流程。

---

## 核心功能

### 新闻自动采集

通过 RSS 持续获取行业资讯，包括：

* AI 行业媒体
* 算力与 GPU 厂商资讯
* 新能源行业媒体
* 机器人行业媒体
* 其他产业链信息源

系统自动抓取最新内容并完成入库。

### AI 新闻理解

针对采集到的新闻内容，调用 DeepSeek API 进行语义分析。

输出统一结构化结果：

```json
{
  "sector": "AI",
  "subSector": "AI Agent",
  "sentiment": "Bullish",
  "summary": "..."
}
```

实现：

* 板块识别
* 细分赛道分类
* 情绪判断
* 中文摘要生成

### 情绪分析

根据新闻内容自动判断市场影响：

* 利好
* 利空
* 中性

并统计形成板块情绪指标。

### AI 情报生成

基于板块内新闻数据自动生成：

* 每日摘要
* 板块总结
* 情绪概览
* 热点动态

降低人工整理成本。

### 板块情报看板

支持查看：

* 新闻数量
* 利好/利空分布
* 市场情绪
* 最新资讯
* 政策动态

帮助用户快速掌握行业整体情况。

```text
     ▼

     │
     ▼

     │
     ▼

     │
     ▼
Supabase
     │
     ▼

     │
     ▼

```

## 项目亮点

### 完整 AI 数据处理链路

实现从资讯采集、语义分析、结构化处理到前端展示的完整流程。

### 非结构化数据结构化

利用大模型将新闻内容转换为：

* 板块信息
* 细分赛道
* 情绪标签
* 摘要内容

便于后续统计与分析。

### 自动化行业情报生成

无需人工筛选资讯即可生成板块级情报信息。

### 轻量级全栈架构

基于 Next.js 与 Supabase 快速构建完整产品原型，实现前后端一体化开发。
